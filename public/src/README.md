# Tukar Wajah Web

Aplikasi web sederhana untuk menukar wajah menggunakan AI (face-api.js).  
Semua proses berjalan di browser tanpa server dan tanpa API key.

## Fitur
- Deteksi wajah otomatis
- Unggah foto sumber dan foto target
- Swap wajah langsung di browser
- Aman, cepat, tanpa backend

## Cara Menggunakan
1. Buka halaman web aplikasi.
2. Upload foto wajah sumber.
3. Upload foto target.
4. Klik tombol "Swap" untuk menukar wajah.

## Teknologi
- HTML, CSS, JavaScript
- face-api.js
- Tailwind CSS (opsional)

## Lisensi
Proyek ini bersifat bebas digunakan dan dikembangkan.
